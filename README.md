# MiniGuia-NotebookLM: Especialista em Desenvolvimento Profissional no LinkedIn e GitHub

Projeto desenvolvido como desafio prático da DIO, utilizando o NotebookLM como ferramenta de aprendizagem ativa para criar um especialista em desenvolvimento profissional focado em LinkedIn, GitHub e carreira em Tecnologia.

## Contexto

Construir uma boa presença profissional na internet tornou-se um diferencial para conquistar oportunidades de estágio, emprego e crescimento na área de tecnologia.

O LinkedIn funciona como um currículo vivo e uma rede de networking, enquanto o GitHub demonstra, na prática, as habilidades técnicas por meio de projetos e documentação.

Neste projeto foi criado um NotebookLM Especialista em Desenvolvimento Profissional, reunindo materiais de referência para atuar como um mentor capaz de orientar melhorias contínuas no LinkedIn, GitHub, portfólio e desenvolvimento de carreira.

## Curadoria de Fontes

Para construir o **NotebookLM Especialista em Desenvolvimento Profissional**, selecionei fontes confiáveis e atualizadas sobre carreira em tecnologia, LinkedIn, GitHub, desenvolvimento profissional e boas práticas de Engenharia de Software. Essas referências foram utilizadas como base de conhecimento para o NotebookLM responder perguntas, gerar resumos e apoiar meu aprendizado.

### Critérios de Seleção

As fontes foram escolhidas considerando os seguintes critérios:

* Conteúdo produzido por empresas e profissionais reconhecidos na área de tecnologia.
* Materiais atualizados e alinhados às práticas atuais do mercado.
* Equilíbrio entre conhecimentos técnicos (Git, GitHub e README), desenvolvimento profissional (LinkedIn e portfólio) e competências comportamentais (soft skills).
* Inclusão de experiências compartilhadas pela comunidade de desenvolvedores para complementar a visão de mercado.

Essas referências serviram como base para a construção do NotebookLM, permitindo gerar resumos, responder dúvidas, comparar diferentes perspectivas e criar um guia de estudos voltado ao desenvolvimento de uma presença profissional sólida no LinkedIn e no GitHub.

### Prompts Reutilizáveis
#### LinkedIn
Analise meu LinkedIn como um recrutador da área de tecnologia e apresente uma lista priorizada de melhorias.

Reescreva meu resumo profissional utilizando storytelling.

Sugira palavras-chave para aumentar minha visibilidade em buscas de recrutadores.

#### GitHub

Avalie meu GitHub segundo boas práticas de Engenharia de Software.

Analise meu README e proponha melhorias.

Quais projetos fariam meu GitHub parecer mais profissional?

#### Carreira

Monte um roadmap para alcançar uma vaga de Desenvolvedor Pleno em cinco anos.

Quais competências técnicas estão faltando no meu perfil?

Analise uma vaga e compare com minhas habilidades.

### Principais Aprendizados

Durante este projeto foi possível compreender que:

a qualidade das respostas depende diretamente da qualidade dos prompts;

fornecer contexto gera respostas mais úteis e personalizadas;

organizar fontes confiáveis melhora significativamente os resultados do NotebookLM;

documentar o processo de experimentação facilita futuras consultas e demonstra maturidade técnica.

### Conclusão

A criação deste NotebookLM mostrou como a Inteligência Artificial pode atuar como uma ferramenta de aprendizagem ativa e apoio ao desenvolvimento profissional. Ao reunir fontes confiáveis, elaborar prompts estratégicos e documentar os resultados obtidos, foi possível construir um material reutilizável para aprimorar continuamente o LinkedIn, o GitHub e o portfólio, fortalecendo a preparação para oportunidades na área de tecnologia.
